# ShopForge Tech Sheets

Professional, local-first PWA for mechanics. NHTSA VIN decode, voice notes + AI polish (BAR compliant), media capture, wear tracking, DOT/PM checklists, professional PDFs.

## Run locally
npm run dev

## Deploy
Import to Vercel for free public https PWA.

## Install on iPhone
Open the deployed URL in Safari > Share > Add to Home Screen.

Data stays on your device.
